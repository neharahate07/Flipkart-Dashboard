📊 FlipMart Sales Dashboard (Power BI Project)
📌 Project Overview

This project presents an interactive Sales Dashboard for FlipMart, designed using Power BI. The dashboard provides meaningful insights into sales performance, customer behavior, and business trends to support data-driven decision-making.

🎯 Objectives
Analyze overall sales performance
Track key business metrics like revenue, profit, and quantity sold
Identify top-performing products and categories
Understand customer purchasing patterns
Enable quick and interactive data exploration

📁 Dataset
The dataset includes information such as:

Order details (Order ID, Date, Customer Name)
Product categories and sub-categories
Sales, Profit, and Quantity
Region-wise and segment-wise data

📊 Dashboard Features
KPIs Overview: Total Sales, Total Profit, Total Orders
Sales Trends: Time-based analysis (monthly/yearly trends)
Category-wise Analysis: Performance by product categories
Region-wise Insights: Sales distribution across locations
Top Products & Customers
Interactive filters and slicers for dynamic analysis

🛠️ Tools & Technologies Used
Power BI Desktop – Data visualization and dashboard creation
Data Cleaning & Transformation – Power Query Editor
DAX (Data Analysis Expressions) – Calculated measures and KPIs

📈 Key Insights
Identified top-performing categories contributing maximum revenue
Observed seasonal trends in sales
Highlighted high-value customers and regions
Found areas with low profit margins for improvement
